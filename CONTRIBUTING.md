# Contributing to IdentityGuard

First off, thank you for considering contributing to this project! 

While this is primarily a **portfolio project** to demonstrate Full Stack AI/ML capabilities, I welcome suggestions and improvements from the community.

## üõ† How to Contribute

### 1. Reporting Bugs
If you find a bug (e.g., the OCR fails on specific lighting conditions), please open an **Issue** and include:
* A description of the bug.
* Steps to reproduce it.
* Your environment (OS, Python version).

### 2. Suggesting Enhancements
Have an idea to improve the Liveness Detection?
* Open an Issue with the tag `enhancement`.
* Explain why this improvement would be valuable.

### 3. Pull Requests (PRs)
I follow standard GitHub flow. If you want to submit code:

1.  **Fork** the repository.
2.  Create a new branch: `git checkout -b feature/AmazingFeature`
3.  Commit your changes: `git commit -m 'Add some AmazingFeature'`
4.  Push to the branch: `git push origin feature/AmazingFeature`
5.  Open a **Pull Request**.

## üìù Coding Standards
* **Python:** Please follow PEP 8 style guidelines.
* **Java:** Follow standard Java naming conventions (CamelCase).
* **Commits:** Write clear, descriptive commit messages.
